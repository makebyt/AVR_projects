Скользящее среднее 
(moving average) — одно из самых простых и часто используемых решений для сглаживания сигнала от АЦП.
Алгоритм просто усредняет несколько последних измерений.

Формула (для N выборок):

Y = (X_0 + X_1 + ... + X_n-1) /N

Простой вариант на псевдокоде (например, для микроконтроллера):

const uint8_t N = 32;      // длина окна усреднения
uint16_t samples[32];      // массив измерений
uint8_t index = 0;
uint32_t sum = 0;

uint16_t moving_average(uint16_t new_value) {
    sum -= samples[index];        // вычитаем старое значение
    samples[index] = new_value;   // записываем новое
    sum += new_value;             // добавляем новое значение
    index++;
    if (index >= N) index = 0;
    return sum / N;               // возвращаем усреднённое
}

Такой способ:

очень прост;

эффективно убирает высокочастотный шум;

имеет плавную реакцию (с задержкой примерно N/2 измерений).


Если нужно менее инерционно — можно взять меньшее N, например 8 или 16.
Если нужно сильнее сгладить шум — увеличить N (32, 64 и т. д.).
