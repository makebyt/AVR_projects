**Volt-Amp meter** <br>
for 12V, 10A charger

- Maximum voltage 20V
- Maximum current 10A

- Voltage and current are displayed alternately
every 3 seconds

- Measurements are taken twice per second

***** Calculations *****

Voltage divider ratio = 18.6
Then 1V = 50 bits

Current sensor gain = 10.75
Then 1A = 100 bits

ADC 1.1V (1023 bits)

**Вольт-Ампер метр** <br>
для зарядного устройства 12в, 10А

- Максимальное значение напряжения 20в
- Максимальное значение токая 10А

- Напряжение и ток отображаются поочередно 
через каждые 3 секунды

- Измерение производится 2 раза в секунду


***** Расчеты *****

Коэффициент делителя напряжения = 18,6
Тогда 1в = 50 бит

Коэффициент усиления датчика тока = 10,75
Тогда 1А = 100 бит

АЦП 1,1в(1023 Бита)
